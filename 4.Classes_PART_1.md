# Constructor
Constructor - A special member function that initializes an object when it is created.
###
It has the same name as Class
###
Constructor is called automatically and with it you can initialize data directly
```cpp
class Box{
  int a,b,c;
  int volume;
  public:
    // 1 way to create constructor
    Box(int height, int length, int width) : a(height), b(length), c(width)
    {//Your code, that compiles when constructor is called}
    // 2 way 
    Box(int height, int length, int width){
    //Your code, that compiles when constructor is called
    a = height;
    b = length;
    c = width;}
};
int main(){
  Box Box1(10,20,30); we set data to a,b,c of Box1

}
```
You also can create mulptiple constructors(like functions overloading)
###
Default that called automatically, and parametrized with objects in it
```cpp
Box() : {std::cout << "Default constructor called";}
Box(int a) : a_side(a) {std::cout << " constructor with 1 var is called";}
Box(int a, int b) : a_side(a), b_side(b) {std::cout << " constructor with 2 vars is called";}
```

# Copy constructor
A special constructor that creates a new object as a copy of an existing object. 
###
ClassName(const ClassName& other);
###
If we pass by value → it would call copy constructor again → infinite loop.
###
Reference avoids extra copy.
###
const prevents modifying the original object.
```cpp
class Student {
private:
    int age;
public:
    // Parameterized constructor
    Student(int a) {
        age = a;
        cout << "Parameterized constructor called" << endl;
    }
    // Copy constructor
    Student(const Student& other) {
        age = other.age;
        cout << "Copy constructor called" << endl;
    }
int main() {

    Student s1(20);     // Calls parameterized constructor
    Student s2 = s1;    // Calls copy constructor. You pass an object by value to a function.You return an object by value from a function.

    return 0;
}
```

# Destructor
Like in new/delete, we need to free our memory space
###
Destructor called automatically, at the end of session
###
~CLASS_NAME(){} Class destructor created
```cpp
class item{
private:
    int number;
    float cost;
public:
// Parameterized Constructor
item(int x, float y)
{
    number = x;
    cost = y;
}
// Destructor
~item(){
    cout << "In Destructor" << endl;}
};
int main()
{
    item i(2, 45.5);   // object created using parameterized constructor
    i.~item();         // explicit call to destructor (NOT recommended)    
    return 0;
}

```
